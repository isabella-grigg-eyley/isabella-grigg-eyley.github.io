Technical Blog 2: Udacity Reflection

What is grid based design?

Grid based design is about structuring webpage elements in the form of rows and columns. These row and column classes have set height/widths, so each class can be applied to the correct column within a row to create consistent heights across variable widths. The example given was dividing the page into 12 columns, allowing the coder to have elements with widths of any factor of 12 (1/12th, 2/12ths, 3/12ths, etc). This makes webpage design and creation easier and more efficient.

What's all the hype about responsive webpages?

Responsive webpages mean you can optimise design across screen sizes and orientations without having static pixel-definition measurements of individual elements.

What is semantic structure?

Semantic structure is HTML structure tags that provide meaning to both the browser parsing the code and people reading it. For example, article tags tell both the browser and coder that the content within the tag is independent and self-contained, like a blog post or newspaper article.
